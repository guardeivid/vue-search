## Vue CLI

Instalar Vue CLI
```sh
npm install -g @vue/cli
```

Crear un proyecto con Vue CLI
```sh
vue create vue-cli-demo
cd vue-cli-demo
```

Ver errores
```sh
npm run lint
```

Lanzar seridor local
```sh

npm run serve
```

Compilar para distribuir
```sh
npm run build
```
